Real Estate Game

How it works

Real Estate Game helps people learn about investing in housing market. User needs to login or sign up to start a game. Every user will start with $1.000.000. User searches for houses using address, city and state. He/she buys a property. The "fast forward 1 month" button creates a bad or good event that affects the property's price. He/she can also sell the property. The ranking in user's profile shows the ranking of players with the most cash once all properties are sold.
